## 快速开始

### 环境要求

- Node.js >= 18
- pnpm >= 10

### 安装依赖

```bash
# 根目录安装所有依赖
pnpm install
```

### 配置环境变量

```bash
# 复制环境变量示例文件
cp .env.example .env

# 编辑 .env 填入实际的 API Key
```

### 运行示例

```bash
# 进入具体示例目录
cd tool-test

# 运行示例
node src/xxx.mjs
```
